# todo-react-django
Simple To-Do application with a DJango + postgres backend and a react frontend.
All packaged with Docker.

## Setup
### Development
run `docker-compose up -d --build`
### Production
run `docker-compose -f .\docker-compose.prod.yml up -d --build`
