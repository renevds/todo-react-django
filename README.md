# todo-react-django
Simple To-Do application with react and Django

## setup   
### For the Django part
- `pip install pipenv`
- `pipenv install`
- `cd /backend`
- `pipenv run manage.py migrate`
- `pipenv run manage.py createsuperuser`   
(to get acces to the admin page)

Develop with: 
- `pipenv run manage.py runserver`

### For the Django part
- `cd /frontend`
- `npm i`

Develop with: 
- `npm start`

### For the React part
